# React + Flask Authentication System

A full-stack authentication mini project built using React.js and Flask.

## Features

- User Signup
- User Login
- Email Validation
- Password Validation
- Confirm Password Validation
- Password Visibility Toggle
- Protected Dashboard
- Logout Functionality
- Toast Notifications
- Responsive UI Design
- Flask REST API Backend

## Technologies Used

### Frontend
- React.js
- React Router DOM
- Axios
- React Toastify
- React Icons
- CSS3

### Backend
- Flask
- Flask-CORS
- Werkzeug Security
